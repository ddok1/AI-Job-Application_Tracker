# AI-Job-Application-Tracker

A Python-based application that helps users organize and manage job applications in one centralized system. The project provides application tracking, analytics, search capabilities, and data export features while following a modular architecture to support scalable and maintainable development.

## Features

* Application management with add, edit, and delete functionality
* Application status tracking for Applied, Interview, Rejected, and Offer stages
* Search and filtering by company, position, and status
* Dashboard analytics with application statistics
* CSV export for application data
* Interactive web interface using Streamlit
* Data persistence using SQLite

## Technologies Used

* Python
* Streamlit
* SQLite
* Pandas
* Object-Oriented Programming (OOP)
* Git and GitHub
